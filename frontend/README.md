# GoVinyl - Frontend

Фронтенд приложения интернет магазина виниловых пластинок на React.

## Функциональность

- 🎵 Каталог виниловых пластинок
- 🔍 Поиск по исполнителю и альбому
- 🏷️ Фильтрация по жанрам
- 🛒 Корзина покупок
- 📦 Управление количеством товаров
- 📱 Адаптивный дизайн
- ✨ Интерактивные компоненты

## Компоненты

### Header
Верхняя навигационная панель с логотипом, меню и кнопкой корзины.

### ProductList
Страница каталога с фильтрацией по жанрам и поиском по названию/исполнителю.

### ProductCard
Карточка отдельной пластинки с изображением, информацией и кнопкой добавления в корзину.

### Cart
Страница корзины с управлением количеством товаров и итоговой суммой.

### Footer
Подвал сайта со ссылками и контактной информацией.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Доступные скрипты

- `npm start` - Запуск в режиме разработки
- `npm build` - Сборка для production
- `npm test` - Запуск тестов
- `npm eject` - Эджект из Create React App (необратимо)
